# Global_Sequence_Alignment_Analyzer

An educational computational biology platform for global sequence alignment using the Needleman-Wunsch algorithm.

## Features
- Global sequence alignment (Needleman-Wunsch)
- Dynamic programming matrix visualization
- Traceback path visualization
- Educational explanations
- Configurable scoring system
- Similarity and Distance optimization modes

## Setup
1. Install dependencies: `pip install -r requirements.txt`
2. Run the application: `python run.py`
