# Project Specifications: Global_Sequence_Alignment_Analyzer

## Technology Stack
- Language: Python 3.12+
- Backend: Flask
- Frontend: HTML/CSS/JS/Jinja
- Database: SQLite

## Core Algorithm
- Needleman-Wunsch Global Alignment
- Configurable Match/Mismatch/Gap scores
- Similarity/Distance optimization modes

## Architecture
- Layered modular design
- Clear separation of algorithm and UI logic

## Git Workflow
- Atomic commits after each meaningful change
- Conventional commit messages
